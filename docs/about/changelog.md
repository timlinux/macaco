# Changelog

All notable changes to MoCaCo will be documented in this file.

## [1.0.0] - 2026-02-21

### Added

- Initial release
- Terminal UI with Bubble Tea and Lipgloss
- 30-task standardized rounds
- Five round types: Beginner, Intermediate, Advanced, Expert, Mixed
- Six task categories: Motion, Delete, Change, Insert, Visual, Complex
- Real-time visual feedback (green/red/amber)
- Comprehensive statistics tracking
- Session and lifetime statistics
- Achievement system
- REST API for server mode
- Client mode for remote play
- Dark, Light, and High Contrast themes
- MkDocs documentation
- Cross-platform support (Linux, macOS, Windows)
- DEB and RPM packages
- Nix flake support

### Vim Features

- Basic motions: h, j, k, l, w, b, e
- Line motions: 0, $, ^
- Find motions: f, t, F, T
- File motions: gg, G
- Delete operations: x, X, d, dd, D
- Change operations: c, cc, C, s, S, r
- Insert operations: i, I, a, A, o, O
- Visual mode: v, V
- Text objects: iw, aw, i", a", i(, a(, etc.
- Yank and put: y, yy, p, P
- Undo/redo: u, Ctrl+r
- Counts: 3w, 2dd, etc.
